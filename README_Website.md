# موقع دكتورة رحمة ميسر

## نظرة عامة
موقع ويب متكامل لدكتورة رحمة ميسر، أخصائية التخاطب وتعديل السلوك.

## هيكل الموقع

### الصفحات الرئيسية
- `index.html` - الصفحة الرئيسية
- `download-app.html` - صفحة تحميل التطبيق
- `thank-you.html` - صفحة شكر بعد التحميل
- `guidance-guide.html` - دليل الإرشادات
- `parent-guide.html` - دليل الوالدين
- `teacher-guide.html` - دليل المعلمين

### الألعاب التعليمية
- `educational-interactive-games.html` - ألعاب تفاعلية تعليمية
- `maze-game-3levels.html` - لعبة المتاهة (3 مستويات)
- `advanced-memory-games.html` - ألعاب الذاكرة المتقدمة
- `find-similarity-game-5levels.html` - لعبة التشابه (5 مستويات)

### المجلدات والملفات

#### /styles/ - ملفات CSS
- `main.css` - الأنماط الرئيسية
- `animations.css` - رسوم متحركة
- `educational-games.css` - أنماط الألعاب التعليمية
- `games.css` - أنماط الألعاب
- `ai-chat.css` - أنماط المحادثة الذكية
- `doctor-photo-effects.css` - تأثيرات صورة الدكتورة

#### /scripts/ - ملفات JavaScript
- `main.js` - الوظائف الرئيسية
- `educational-games.js` - منطق الألعاب التعليمية
- `games.js` - منطق الألعاب
- `ai-chat.js` - المحادثة الذكية
- `animations.js` - الرسوم المتحركة

#### /dr_rahma_website/
- `imgs/dr_rahma_photo.jpg` - صورة دكتورة رحمة

#### /game-assets/
- `images-map.json` - خريطة الصور للألعاب

#### التطبيق
- `Rahma Maysar App.aab` - ملف التطبيق

## التشغيل
1. افتح `index.html` في المتصفح
2. الموقع يعمل بدون خادم محلي
3. جميع الملفات والأصول مضمنة

## المتطلبات
- متصفح ويب حديث يدعم HTML5 و CSS3
- اتصال إنترنت للخطوط الخارجية (اختياري)
